## Top actions (ranked by impact)

1. **Fix the `lib/` → `components/`/`app/` layer inversion** (rows 1–5). Centralize `ArtifactKind`, `VisibilityType`, `UserType`, `MessageResponse` in `lib/types.ts`. Highest structural impact; unblocks clean testing of `lib/`. Hand off to `aidd-dev:07-refactor`.
2. **Bound the unbounded DB queries** (rows 9–11). Add `limit`/pagination to `getMessagesByChatId`, `getVotesByChatId`, `getDocumentsById` before chat histories grow. Hand off to `aidd-dev:07-refactor`.
3. **Lock down file uploads** (row 6). Public-by-default Blob storage with no expiry is the most material security exposure. Hand off to `aidd-dev:07-refactor`.
4. **Add a unit-test layer for `lib/`** (row 21). The suite is 100% e2e; pure logic in `lib/` is untested. Hand off to `aidd-dev:06-test`.
5. **Tighten input schemas** (rows 7, 8). Replace `z.record(z.unknown())` and unconstrained URL validation with strict schemas. Hand off to `aidd-dev:07-refactor`.
