# VCS

## Setup

- Main branch: `main`
- Platform: GitHub (`Dada342/aidd-chat`)
- Ticketing: none (bootcamp project)

## Branches

- Format: `type/short-description`
- Types in use: `feat`, `fix`, `chore`, `refactor`

## Commits

- Convention: Gitmoji + feature tag
- Format: `emoji [feature-name] file: description`
- Examples: `✨ [chat] mistral: add direct provider`, `🔧 [chat] .gitignore: exclude sessions`
- Rules: imperative mood, no Co-Authored-By, no force push
